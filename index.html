Tome

A static site generator for Drupal 9

Use tools you love

Making a static site with Tome is as easy as making any Drupal site. Create a theme, create content, and push static HTML to production.

Commit your content

With Tome, your content is stored as static JSON files. Make a change, commit it, and spin down Drupal.
